# Objects

```javascript
const obj = {
	a: foo
	b: bar
}

console.log(obj.a) // foo
Object.keys(obj) // ["a", "b"]
```

_Define with a variable_

```javascript
let key = foo;
let obj = { [key]: bar }; // {foo: bar}
```
